# React Native Demo

## Note 
    一个用react native的类似笔记本的Demo

### 安装
1. clone repo
    git clone ... .git
2. cd 项目路径
3. 安装依赖  `npm install `
4. 安装以Android为例，
    a：首先使用`adb reverse tcp 8081 tcp:8081` 将电脑8081端口反向代理带测试机
    b:`react-native run-android`  运行
